RWXBioFuge
==========

RWXBioFuge code for operating an Open Source microcentrifuge.

WARNING: Fast spinning rotors are dangerous!

Input:
- Time rotary encoder
- RPM rotary encoder
- Start button
- Short run button
- Lid detector
- Infrared RPM sensor

Output:
- Pulses for Electronic Speed Controller
- Serial USB
- Ethernet
- I2C 16x2 LCD display
