RWXBioFuge
==========

RWXBioFuge code for operating an Open Source microcentrifuge.

Input:
- Time potentiometer
- RPM potentiometer
- Start button
- Short run button
- Lid detector
- Infrared RPM sensor

Output:
- Pulses for Electronic Speed Controller
- Serial USB
- Ethernet
- I2C 16x2 LCD display
